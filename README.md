
## JavaScript Background Color Changer 

---

** Ye ek beginner-friendly JavaScript project hai jisme user alag-alag color buttons par click karke webpage ka background color change kar sakta hai.**

---

## PROJECT HIGHLIGHTS
- Simple & Clean UI  
- Fast Color Switching  
- Beginner Friendly Logic  
- Pure HTML, CSS & JavaScript  

---

## AVAILABLE COLORS
- Red  
- Blue
- Black  
- Yellow
- Green  
- Indigo
- Olive  
- Tan  

 **Button par click karte hi background instant change ho jata hai**

---

##  TECHNOLOGIES USED
- **HTML** – Page Structure  
- **CSS3** – Styling & Layout  
- **JavaScript (DOM)** – Color Change Logic  

---

## PROJECT STRUCTURE

---

## HOW TO USE
 Project folder download ya clone karo  
 `index.html` file browser me open karo  
 Kisi bhi color button par click karo  
 Background color change hota dekho
